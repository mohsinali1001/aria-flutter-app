# ARIA – AI Personal Secretary

ARIA is a conversational AI assistant mobile application that helps users manage emails, calendar events, and reminders through a chat interface.

## Technology Stack

* Flutter (Mobile App)
* n8n (Backend Workflows)
* Supabase (Database)
* Firebase (Notifications)
* Gemini / Groq AI Models

## Features

* Email Summary
* Draft Email Reply
* Send Email
* Day Brief
* Set Reminder
* View Reminders
* Email Search
* Calendar View
